# rpg.pan.earth

Repository for **Thicket Web Mind (TWM)** — a runtime-born methodology of local truth, executable beings, maintained flow, and controlled becoming.

This release is offered for **club reading**: human discussion and AI-assisted critique, extraction, and adversarial clarification.
